# Gestor de Tareas - Semana 2

Proyecto desarrollado en Java aplicando Programación Orientada a Objetos.

## Funcionalidades
- Agregar tareas
- Listar tareas
- Marcar tareas como completadas
- Eliminar tareas
- Mostrar estadísticas

## Estructura
- modelo: clase Tarea
- servicio: clase GestorTareas
- Main.java: menú interactivo

## Ejecución
javac Main.java modelo/.java servicio/.java
java Main
