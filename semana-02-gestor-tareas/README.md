# Gestor de Tareas – Semana 2

Proyecto desarrollado en Java como parte de la materia Programación Orientada a Objetos.

## Funcionalidades
- Agregar tareas con ID automático
- Listar todas las tareas
- Listar tareas pendientes y completadas
- Completar tareas
- Eliminar tareas
- Mostrar estadísticas

## Compilación
```bash
javac Main.java modelo/.java servicio/.java
