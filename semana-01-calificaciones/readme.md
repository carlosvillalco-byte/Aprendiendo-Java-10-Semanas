# Semana 1: Sistema de Calificaciones
Programa que gestiona notas de 5 estudiantes .
## Funcionalidades
- Almacena 5 estudiantes con 3 notas cada uno
- Calcula promedios individuales y general
- Determina aprobados y reprobados
- Muestra nota maxima y minima
## Como ejecutar
1. Entrar a la carpeta : ‘cd semana -01 - calificaciones ‘
2. Compilar : ‘javac Main .java ‘
3. Ejecutar : ‘java Main ‘
